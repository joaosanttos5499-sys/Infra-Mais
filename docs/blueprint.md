# **App Name**: CityFix

## Core Features:

- Report Creation: Allow users to create detailed reports about infrastructure problems including text description, location and photos.
- Geolocation: Automatically detect the user's current location or allow them to manually specify the location of the infrastructure problem using a map.
- Report Categorization: Categorize reports based on the type of infrastructure problem (e.g., potholes, broken streetlights, water leaks).
- AI Report Summary: Use AI to generate a short summary of the problem reported based on user's input and pictures using a tool. The city employee is presented with the short summary.
- Report Submission: Submit reports to the city hall.
- Issue tracking: Allow the city employee to update the status of the issue reported.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and reliability.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Teal (#008080) for interactive elements and important calls to action.
- Font pairing: 'PT Sans' (sans-serif) for both headlines and body text.
- Use clear, recognizable icons for different categories of infrastructure problems.
- Implement a clean and intuitive layout that prioritizes ease of navigation.
- Use subtle animations for feedback on user actions and loading states.
